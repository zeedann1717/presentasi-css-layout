# Arsitektur Visual: CSS Positioning & Layouting

Halo! Selamat datang di repository ini. 

Repository ini berisi file presentasi PDF yang membahas tuntas tentang arsitektur visual, khusus fokus pada tata letak (layouting) dan posisi elemen menggunakan CSS. Materi ini disusun dari evolusi awal hingga tren tahun 2026 untuk menciptakan antarmuka yang presisi dan responsif.

## Topik Pembahasan
Di dalam presentasi ini, kamu akan menemukan materi tentang:
- **Evolusi & Dasar CSS:** Filosofi Box Model dan penggunaan unit pengukuran utama (PX vs REM).
- **Logika Positioning:** Penggunaan tipe display, serta pemahaman posisi dari static hingga sticky dan dimensi ketiga Z-Index.
- **Flexbox Mastery:** Menguasai tata letak satu dimensi, pengenalan Container vs Items, serta aturan Justify, Align, dan pembungkusan elemen.
- **CSS Grid Era:** Merancang arsitektur dua dimensi dan pengenalan fitur pewarisan tata letak lewat Subgrid 2026.
- **Strategi Responsif 2026:** Praktik terbaik desain multi-platform menggunakan kombinasi Media Queries dan Container Queries.

##  Cara Mengunduh Materi
Karena GitHub tidak bisa memutar file presentasi secara langsung, kamu bisa mengunduh file ini untuk dibaca secara *offline*:

2. Klik tombol **Download** (ikon panah ke bawah) di pojok kanan atas tampilan file.
3. Buka file tersebut menggunakan Microsoft PowerPoint atau aplikasi pembaca slide lainnya.
